# Сапёр (Minesweeper) — браузерная мини-игра

Классический **сапёр** на чистом **HTML, CSS и JavaScript**: без сборки и фреймворков. Интерфейс на русском, три уровня сложности (8×8, 9×9, 12×12), таймер, счётчик мин, первая открытая клетка всегда безопасна.

## Как запустить локально

Откройте `index.html` в браузере или поднимите любой статический сервер в этой папке:

```bash
npx serve .
```

## GitHub Pages

1. Загрузите содержимое этой папки в отдельный репозиторий (например `minesweeper-game`).
2. **Settings → Pages →** ветка `main`, папка `/ (root)`.
3. Игра будет доступна по адресу `https://<user>.github.io/minesweeper-game/`.

## Управление

- **Левый клик** — открыть клетку  
- **Правый клик** — флаг  
- На сенсорных экранах: **долгое нажатие** — флаг  

## Структура

| Файл       | Назначение        |
|-----------|-------------------|
| `index.html` | Разметка, сайдбар |
