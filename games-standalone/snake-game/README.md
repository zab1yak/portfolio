# Змейка (Snake) — браузерная мини-игра

**Змейка** на **HTML5 Canvas**, **CSS** и **JavaScript**: плавное движение, пауза, счёт и сохранение лучшего результата в `localStorage`. Управление с клавиатуры, интерфейс на русском.

## Как запустить локально

Откройте `index.html` в браузере или:

```bash
npx serve .
```

## GitHub Pages

1. Залейте файлы в репозиторий (например `snake-game`).
2. **Settings → Pages:** источник `main`, папка `/ (root)`.
3. URL: `https://<user>.github.io/snake-game/`.

## Управление

Стрелки или **W A S D** — направление, **Пробел** — пауза / продолжить.

## Файлы

| Файл        | Назначение      |
|------------|-----------------|
| `index.html` | Разметка, canvas |
| `style.css`  | Оформление       |
| `game.js`    | Игровой цикл     |
