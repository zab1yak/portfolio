# Астероиды (Asteroids) — браузерная мини-игра

Аркада **«Астероиды»** на **Canvas** и **JavaScript**: корабль, выстрелы, волны астероидов, щит и игровая статистика. Оформление и подсказки на русском.

## Как запустить локально

Откройте `index.html` в браузере или:

```bash
npx serve .
```

## GitHub Pages

1. Создайте репозиторий (например `asteroids-game`) и загрузите файлы из этой папки.
2. **Settings → Pages:** ветка `main`, корень сайта.
3. Демо: `https://<user>.github.io/asteroids-game/`.

## Управление

Смотрите подсказки на экране игры (обычно стрелки / поворот, ускорение, стрельба — в зависимости от версии в `game.js`).

## Файлы

| Файл        | Назначение        |
|------------|-------------------|
| `index.html` | Разметка, canvas   |
| `style.css`  | Стили интерфейса   |
| `game.js`    | Физика и геймплей  |
