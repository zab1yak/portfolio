## Что внутри проекта

- Статический сайт: **HTML**, **CSS**, **JavaScript** (без React/Vite — открывается везде).
- **Русский и английский** интерфейс (переключатель RU/EN, язык запоминается в браузере).
- Тёмная / светлая тема, плавный скролл по якорям, экран загрузки, фон с лёгкой анимацией.
- Форма «Написать мне» через [FormSubmit](https://formsubmit.co) на `zab1rabota@gmail.com` (при первой отправке с сайта подтверди почту по письму от сервиса).

### Структура

```
├── index.html
├── css/style.css
├── js/main.js      # логика: тема, меню, скролл, язык
├── js/i18n.js      # тексты RU/EN
└── README.md
```

### Локально

```bash
npx --yes serve .
```

Открой в браузере адрес, который покажет `serve`, или просто `index.html`.
